Project Name-Intelliquiz: Quiz System with Difficulty Adaptation  

 The system stores quiz questions with difficulty levels.  
• Question difficulty changes based on user performance.  
• The quiz uses a simple Java Swing interface.  
• User actions are handled using event handling.  
• Each question has a fixed time limit.  
• User performance is tracked during the quiz.  
• Final result and feedback are shown after completion 

Technology Stack  
Programming Language: Java  
GUI Framework: Java Swing  
IDE: IntelliJ IDEA 
Concepts:  
• Object-oriented programming for structure  
• Multithreading for the timer  
• Event handling for user actions   
• Java collections to store questions and performance data.
